# 08 - Git

Version control with Git — from basics to advanced workflows.

## Chapters

### 01 - Basics
| Topic | Description |
|-------|-------------|
| init-clone | Create repos, clone from remote |
| add-commit | Stage changes, write commits |
| status-log | Inspect working tree and history |
| diff-show | Compare changes between commits |

### 02 - Branching
| Topic | Description |
|-------|-------------|
| branches | Create, switch, delete branches |
| merging | Merge strategies, resolve conflicts |
| rebasing | Rebase workflows, interactive rebase |
| cherry-pick | Apply specific commits to other branches |

### 03 - Remote
| Topic | Description |
|-------|-------------|
| remote-repos | Add, remove, inspect remotes |
| fetch-pull-push | Sync with remote repositories |
| pull-requests | PR workflow, code review best practices |
| fork-workflow | Fork, upstream sync, contributing to OSS |

### 04 - Advanced
| Topic | Description |
|-------|-------------|
| stash | Save and restore work in progress |
| reset-revert | Undo changes safely (reset vs revert) |
| bisect | Binary search for bug-introducing commits |
| hooks | Pre-commit, pre-push, commit-msg hooks |
| submodules | Nested repositories |
| worktrees | Multiple working directories from one repo |

## Suggested Order

```
01-basics --> 02-branching --> 03-remote --> 04-advanced
```
